# КПТ-Курс — наркологія

PWA-застосунок для проходження персонального 16-тижневого курсу когнітивно-поведінкової терапії (наркологічний профіль): [kpt-kurs.md](kpt-kurs.md).

## Що вміє

- **Блоки курсу (0–6)** — теорія, дриль, пастки, цільові пункти CTS-R, чеклист критерію виходу.
- **Оцінка знань** — 112 питань (ситуаційні вправи включно), поріг складання 80%, помилки повертаються в режимі «Повторення».
- **Журнал сесій** — 5-рядковий формат курсу + бали CTS-R (0–6) + MITI-лічильник з автоматичною перевіркою порогів (рефлексії:питання ≥1:1, складні ≥40%, нуль MI-неузгоджених).
- **Динаміка** — графіки балів за пунктами CTS-R, підсвічування пунктів, що не ростуть.
- **Бекап** — експорт/імпорт JSON, експорт журналу в Markdown (для Obsidian).

## Приватність

Усі дані (журнал, прогрес, результати тестів) зберігаються **лише локально на пристрої** (localStorage). Жодного сервера, акаунтів чи синхронізації немає. Перед очищенням даних браузера робіть бекап (Ще → Бекап).

## Встановлення на телефон

1. Відкрийте сайт застосунку в браузері телефона.
2. **Android (Chrome):** меню ⋮ → «Додати на головний екран» / «Встановити застосунок».
3. **iPhone (Safari):** кнопка «Поділитися» → «На початковий екран».

Після встановлення застосунок працює офлайн.

## Локальний запуск
